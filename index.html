🔒 보안 & 개인정보

  • ✅ 모든 계산은 클라이언트(브라우저)에서만 실행됩니다.
  • ✅ 입력한 금액, 날짜, 금리 등 어떤 데이터도 서버로 전송되지 않습니다.
  • ✅ 외부 API 호출이 없습니다. (JavaScript 라이브러리 로드 없음)
  • ✅ GitHub Pages로 호스팅 시 HTTPS가 기본 적용됩니다.
  • ✅ URL 파라미터로 공유할 수 있으며, URL에 포함된 정보는 브라우저 주소창에만 존재합니다.
  • ⚠️ URL을 공유하면 입력값이 노출됩니다. 민감한 실제 데이터는 공유 시 주의하세요.
